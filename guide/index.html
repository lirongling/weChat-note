# 第一个项目总结

# 登录注册

# 收获

  1. 掌握了post请求(第一个请求数据的jiekoulujing,第二个是请求的参数)

    this.$axios    .req("接口路径", "参数")    .then(res => { })
                    .catch(err => {
                        console.log(err);
                    });

  2. 掌握了element(tabs)切换

  3. from表单的验证

# 遇到的问题

  1. 把定义的数据写到from表单里去,不够仔细,结果导致进行组件传值没有效果

    ruleForm: { pass: '', username: '', checkPass: '',activeName0:'login',},

# 首页

# 收获
  1. 学会使用v-charts(vue图表库),这个第三库还是很重要的
  2. 学会使用element 标签
  3. <div class="top-status"> <el-tag v-if="item.status===1" type="success" plain>success</el-tag> <el-tag v-if="item.status===0" type="warning" plain>pending</el-tag></div>
  4. 学会使用element NavMenu 导航菜单组件
# 遇到的问题

  1. 刚开始对vue图标库使用的不熟练

  2. NavMenu 刷新后点击效果消失了

    :default-active="$route.path"

# 导出Excel

# 收获
  1. 使用table表格
  2. 熟练使用分页功能
  3. 同时也学会了用vue-json-excel导出Excel
# 遇到的问题

  1. 如果不在第一页搜索出现显示没用数据,后来在watch监听输入框将当前页置为1

    inputText(val) { // 页数重置为1 this.currentPage = 1;}

  2. 在进行修改数据时对输入框里的数据处理不够严谨(比如修改的现价只能是数字),后使用表单进行验证

# 上传文章

# 收获
  1. 熟练使用mavon-editor(markdown编辑器)
  2. 也对from表单验证更熟悉
# 遇到的问题

  1. 对时间选择的处理,不能选择将来的时间

    disabledDate(time) { return time.getTime() >= Date.now()},

# 已发布

# 收获

  1. 使用table表格的插槽

    <template slot-scope="scope"> <div class="flex"> <el-button type="primary" @click="edit(scope.$index, scope.row)">编辑</el-button> <el-button type="danger" @click="del(scope.$index, scope.row)">删除</el-button> <el-button type="info" @click="see(scope.$index, scope.row)">查看</el-button> </div></template>

  2. 熟练使用带参跳转

  3. 熟练使用分页表格

# 遇到的问题
  1. 发现不能在发布文章页就格式化时间,这样传的格式化后时间就不再是Date类型,就不能存进去

# 标签页

# 收获
  1. 进一步熟练使用tabs切换
# 遇到的问题
  1. 代码还不够精简

# 统计

# 收获
  1. 进一步使用v-charts(vue图表库)
# 遇到的问题
  1. 分析的不够彻底，有时候把问题复杂化

# 上传图片

# 收获
  1. 学会使用element 的上传图片功能
  2. 学会使用vue-image-crop-upload上传图片
# 遇到的问题

# 总结

基本功能都已实现,但还不够完善,有些界面不够美观,代码仍然还可以精简,还有就是注释必须刚开始就应该加上,这次很多注释都是后面才加上的,对代码的规范应该加强,比如说命名,该抽分组件就应该抽分成组件.
